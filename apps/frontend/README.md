# Price Insight Frontend

Nuxt 4 frontend for Price Insight with:

- `@nuxt/ui`
- Tailwind CSS v4
- `nuxt-auth-utils`
- Google OAuth login

## Setup

Create a `.env` file from `.env.example`, then install dependencies and run:

```powershell
npm install
npm run dev
```

Google OAuth callback URL:

```text
http://localhost:3000/auth/google
```
